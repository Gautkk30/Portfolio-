## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Gautkk30/<repository-name>.git
```

Navigate to the project:

```bash
cd <repository-name>
```

Install dependencies (if applicable):

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
